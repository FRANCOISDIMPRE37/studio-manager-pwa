🔐 Se connecter

Accéder à votre espace studio

1 Ouvrez Safari sur l'iPad et allez sur studio.studiomanagereurope.eu
2 Tapez votre code PIN à 4 chiffres sur le clavier à l'écran
3 Vous arrivez directement sur le Tableau de bord
💡 Vous pouvez aussi utiliser email + mot de passe en appuyant sur "Se connecter avec email →"

👤 Créer un client

Enregistrer un nouveau client

1 Appuyez sur 👥 Clients dans le menu gauche
2 Appuyez sur le bouton + Nouveau client
3 Remplissez : Nom, Prénom, Date de naissance, Téléphone, Email
4 Appuyez sur Enregistrer
💡 Le numéro client est généré automatiquement. Vous pouvez aussi importer des clients depuis un fichier Excel.

📋 Remplir une fiche

Créer un document pour un client

1 Appuyez sur 📋 Documents dans le menu
2 Choisissez le type de fiche : Piercing, Tatouage ou Dermographie
3 Recherchez et sélectionnez le client dans la liste
4 Remplissez tous les champs marqués * (obligatoires)
5 Faites signer le client (voir slide suivant)
6 Appuyez sur ✓ Sauvegarder le document

✍️ Faire signer le client

Signature électronique obligatoire

1 Faites défiler la fiche jusqu'en bas pour voir la zone de signature
2 Tendez la tablette au client
3 Le client signe avec son doigt dans la zone prévue
4 Appuyez sur ✅ Valider pour confirmer la signature
5 Appuyez sur ✓ Sauvegarder le document
⚠️ La signature est OBLIGATOIRE pour les questionnaires médicaux. Sans signature, le document ne peut pas être sauvegardé.

💉 Fiches Piercing

Documents disponibles pour le piercing

01 Questionnaire Médical Mineur + Autorisation Parentale
02 Questionnaire Médical Majeur
03 Fiche de Traçabilité Matériel Stérile
A-G Soins Post-Piercing : Oreilles, Nez, Labret, Nombril, Téton, Arcade, Surface
💡 Les fiches de soins (A à G) peuvent être imprimées et remises au client après le piercing.

🖨️ Imprimer une fiche

Imprimer ou enregistrer en PDF

1 Ouvrez la fiche du client (Clients → nom → fiche)
2 Appuyez sur le bouton 🖨️ Aperçu / Imprimer en haut
3 Choisissez : Imprimer (imprimante) ou Enregistrer en PDF
4 Sélectionnez votre imprimante AirPrint et appuyez sur Imprimer
💡 Le logo et les informations de votre salon apparaissent automatiquement sur chaque fiche imprimée.

📁 Archives Numérisées

Numériser les anciens dossiers papier

1 Menu gauche → Archives Numérisées
2 Appuyez sur + Nouveau en haut à droite
3 Sélectionnez la date de visite du client
4 Remplissez le Nom et Prénom du client
5 Photographiez le dossier papier avec 📷 Ajouter des photos
6 Appuyez sur ✓ Sauvegarder
🔴 RGPD : Les dossiers sont automatiquement marqués expirés après 3 ans.
